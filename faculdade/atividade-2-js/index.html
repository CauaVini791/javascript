Número aleatório

Criar um programa JS que sorteie um número aleatório.
O usuário deve entrar com um número e o programa deve verificar as seguintes condições:
1- Se o número for menor que 1, maior que 10 ou não for um número, o programa deverá mostrar um alerta informando ao usuário entrar com um número válido;
2- Se o número que o usuário escolher for menor que o número sorteado, um alerta deverá ser exibido informando que o número é maior;
3- Se o número que o usuário escolher for maior, um alerta deverá informá-lo para escolher um numero menor;
4- Enquanto o número não for escolhido corretamente, o usuário terá possibilidade de tentar várias vezes;
5- Após escolhido o número correto, deve ser apresentado na página HTML a quantidade de tentantivas até o número ter sido adivinhado.
